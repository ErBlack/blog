№50

Типи — это такая индейская палатка. Она состоит из прямых палок, составленных домиком, и натянутых вокруг шкур или парусины. В середине конструкции есть отверстие для выхода дыма, поэтому внутри можно разводить костёр.

Чтобы нарисовать типи, нужно начать с двух пересекающихся сверху палок. Затем аккуратно добавить завёрнутую вокруг них ткань, чтобы было видно отверстие для дыма. Далее нарисовать нижнюю часть, чтобы выглядело, как будто это палки, обёрнутые тканью. И добавить верёвку, которая держит всё это сверху.
